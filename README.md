# Brain_Metastases_ecDNA

This repository contains R scripts used to analyze ecDNA-related genomic features, paired primary-metastatic dynamics, and survival associations in brain metastasis samples.

## Repository structure

```text
.
├── Figure1-git.R
├── Figure2-git.R
├── Figure3-git.R
├── Figure5-git.R
├── Inputs/
└── Outputs/
```
Raw patient-level data are not included in this repository because of data-sharing and privacy restrictions. To run the scripts, place the required input files under Inputs/.

## Running the analysis
Run the scripts from the repository root:
```text
Figure1_Atlas.R
Figure2_genomics.R
Figure3_Paired_ecDNA.R
Figure5_survival.R
```
Recommended order:
```text
Figure1_Atlas.R
Figure2_genomics.R
Figure3_Paired_ecDNA.R
Figure5_survival.R
```
`Figure5-git.R` uses processed outputs generated by `Figure2-git.R`.

## Inputs
The scripts expect input files under the `Inputs/` directory. Main input types include:
- sample metadata workbook
- WES GCAP focal CNA records
- WGS gene-level ecDNA / amplicon annotations
- mutation files
- CNVkit segment files
- feature-level copy-number tables
- oncogene reference list
Excel sheet names in the scripts must match the actual sheet names in the metadata workbook.

## Outputs
All results are written to `Outputs/`.
```text
Outputs/Figure1_Atlas/
Outputs/Figure2_genomics/
Outputs/Figure3_Paired_ecDNA/
Outputs/Figure3_AllPaired/
Outputs/Figure5_survival/
```
Each output folder contains figure panels, intermediate RDS files, and summary tables generated by the corresponding script.

## Notes
- Scripts assume they are run from the repository root.
- Input paths can be modified at the top of each script.
- Raw clinical and sequencing data are not distributed with this repository.
- Generated results may vary if input files or metadata annotations are updated.
